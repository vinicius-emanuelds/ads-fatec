# Backup e Manutenção

## **Backup Manual** <a href="#id-71-backup-manual" id="id-71-backup-manual"></a>

* Acesse o diretório `databases/`.
* Faça cópias dos arquivos `salas.csv` e `usuarios.csv` para um local seguro.



## **Atualizações e Suporte** <a href="#id-72-atualiza-c3-a7-c3-b5es-e-suporte" id="id-72-atualiza-c3-a7-c3-b5es-e-suporte"></a>

*   Atualize o sistema com:

    ```bash
    git pull origin main
    ```



* Relate problemas no repositório GitHub do projeto na aba _Issues_.
