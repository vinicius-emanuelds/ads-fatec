# PRINCIPAIS ATALHOS NO VSCODE
* ALT + ↑ (ou ↓) = Move a linha selecionada
* SHIFT + ALT + ↑ (ou ↓) = Copia a linha selecionada
* CTRL + SHIFT + K = Apaga a linha selecionada
* CTRL + D = Multi-seleção de ocorrências
* ALT + CLICK = Multicursor
* CTRL + K e depois Z = Zen mode (para sair, ESC)
* CTRL + ESPAÇO = Autopreenchimento
