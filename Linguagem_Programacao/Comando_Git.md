
| Comando | Descrição |
| --- | --- |
| git clone | Clona um repositório Git existente para um novo diretório (pasta)local |
| git commit | Grava alterações no repositório |
| git pulll | “Puxa” as alterações do repositório remoto para o local (busca e mescla) |
| git push | “Empurra” as alterações do repositório local para o remoto |
| git config --global [user.name](http://user.name/) | Definir nome |
| git config --global user.email | Definir e-mail |
| git config --global init.defaultBranch | Definir nome padrão da Branch |
| git config --global --list | Exibe as configurações já ajustadas |
| mkdir (escolha um nome) | Criar diretório |
| git init | “Transforma” repositório local em diretório Git |
| git clone (url) [nome] | Clona o repositório, alterando o nome da pasta |
| git status | Mostra o status do arquivo |
| touch | criar arquivos md |
| rm -rf .git | remove o a “pasta” .git |
| git restore | restaura o arquivo |
| git commit —amend -m”MENSAGEM” | alterar a mensagem do commit |
| git reset --soft [cash do commit] | adiciona os arquivos do último commit na área de preparação |
| git log | gera um log das alterações no repositório |
| git reflog | log detalhado das alterações |
| git reset [cash do commit] 
git reset --mixed [cash do commit] | adiciona os arquivos posteriores ao commit indicado na árvore de trabalho |
| git reset --hard [cash do commit] | apaga todos os arquivos posteriores ao commit indicdo |
| git checkout - b | cria uma nova branch e troca o “apontamento” para ela |
| git branch  -v | lista o último commit de cada branch |
| git merge [nome] | Mescla a branch com a branch principal (main) |
| git branch | lista as branches no repostiório |
| git branch -d [nome] | deleta a branch |
