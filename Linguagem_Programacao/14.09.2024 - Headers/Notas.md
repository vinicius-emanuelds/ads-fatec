# Funções

## Sintaxe
```
tipo_de_retorno nome_da_funcao(parametros){
    //Corpo da função
    ...
    return valor;
}
```

## Protótipo de Função
```
tipo_de_retorno nome_da_funcao(parametros);
```
## Procedimento
Não tem retorno

|  | **Tipo de Retorno** | **Return**|
| --- | --- | --- |
| **Sintaxe** | variável = getchar() | scanf("%c",&variável) |
| **Entrada** | Lê um único caractere | Lê a entrada formatada |
| **Saída**| Retorna um caractere com valor inteiro <br> Se não houver mais caracteres, retorna EOF (End Of File)|

# Bloco
Coleção de zero ou mais declarações agrupadas entre chaves {}
- Estruturas de controle: múltiplas instruções realizadas
- Escopo: quando as variáveis declaradas dentro do bloco, o escopo é local ao bloco
