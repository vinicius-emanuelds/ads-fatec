# getchar x scanf
|  | **getchar** | **scanf**|
| --- | --- | --- |
| **Sintaxe** | variável = getchar() | scanf("%c",&variável) |
| **Entrada** | Lê um único caracter | Lê a entrada formatada |
| **Saída**| Retorna um caracter com valor inteiro <br> Se não houver mais caracteresa, retorna EOF (End Of File)|

# Bloco
Coleção de zero ou mais declarações agrupadas entre chaves {}
- Estruturas de controle: múltiplas instruções realizadas
- Escopo: quando as variáveis declaradas dentro do bloco, o escpopo é local ao bloco
